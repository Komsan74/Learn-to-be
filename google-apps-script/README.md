# Google Apps Script

อีกขั้นหนึ่งของการใช้งานกูลเกิ้ลสเปรดชีตให้มีประสิทธิภาพสูงสุด ก็คือการพัฒนาฟังชั่นในการคำนวนต่าง ๆ รวมไปถึงการเข้าถึงข้อมูลในสเปรดชีตผ่าน Google Appscript ในรูปแบบต่าง ๆ ตลอดจนการพัฒนาให้เป็นเว็บแอพพลิเคชั่นโดยใช้สเปรดชีตเป็นฐานข้อมูล ซึ่งผู้บันทึกจะนำมาเสนอเป็นหัวข้อให้ได้ติดตามกันต่อไป
